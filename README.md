# mad-city-rob-script
A script you can implement inside your script to rob places in Mad City (Chapter 1)!

# How To Use
1. Run this code in your executor:
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/lunar0x4/mad-city-rob-script/refs/heads/main/rob.lua"))()
```

2. Go ahead and run any of these once it's loaded:
```lua
rob("Bank") -- Instead of bank, you can use these: "Plane", "Bank", "Jewelry", "Airport", "Casino", "Club", "Mall", "ComputerStore", "Cruise", "ATM"
```

3. Let the script rob!

## Notes
- It is queueable (so you can define rob("Bank") or rob("Jewelry") at the same time and it will queue both of them)
